# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO

O Lab explica como que depois de ter criado a conta gratuita, fazer o acesso inicial ao Portal do Azure, para explorar os recursos onde terei o mesma visualização que uma empresa grande, o portal é o mesmo para todos.

Aprendemos a configurar o idioma e o Tema das cores. E no menu "Todos os Serviços" podemos ver a lista organizada por categorias: Computação (máquina virtual), Rede (firewall), Armazenamento (migração).

Vimos por último sobre a versão prévia, onde o serviço ainda está em teste, sem SLA (garantia).
